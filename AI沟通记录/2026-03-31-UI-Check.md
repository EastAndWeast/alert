## AI 行径与分析记录
### 1. Hover tooltip 显示为残缺 HTML 的溯源修复：
- 经过检索 `title="` 拼接的地方，我发现 `js/modules/rules.js` 中 **近期告警** 表格在绑定 tooltip 属性时存在漏洞：
  ```javascript
  html += '<td class="trigger-value-cell" title="' + triggerValue + '">' + triggerValue + '</td>';
  ```
  由于 `triggerValue` 返回自带 `<span style="...">` 格式的话，原生浏览器的 `title` 将遭受 HTML 标签断点异常（字符串内部包裹了双引号）。这导致最终生成的代码出现乱码 `">Day PnL...`。
- **修复措施**：已经使用原生正则脱取方法 `triggerValue.replace(/<[^>]+>/g, '').replace(/"/g, '&quot;')` 处理成纯文本，再赋给 `title`。现原生 Hover 显示能够完美输出类似 `Day PnL $-15,000 | 报警限额 -10000` 这样的清晰细节。

### 2. 左侧侧边栏四页面功能一致性分析：
