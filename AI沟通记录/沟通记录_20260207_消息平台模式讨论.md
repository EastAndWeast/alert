# 沟通记录 - 2026-02-07 消息平台推送模式讨论

## 参与者
- 用户
- AI (Antigravity)

## 讨论内容

### 用户要求
1. **统一性**：Teams, Lark, Telegram, Slack 四个平台的消息模板需尽量统一。
2. **直观性**：信息展示需包含发起者、时间、触发条件、触发规则等。
3. **可操作性**：包含跳转按钮，直接链接到 Web 网站对应页面。
4. **当前阶段**：仅进行分析设计，不涉及具体实施。

### AI 建议与分析
- **统一方案**：建议采用中性化数据 Schema (JSON)，再根据各平台的 UI 规范（Teams: Adaptive Cards, Lark: Interactive Cards, Slack: Block Kit, Telegram: HTML/Inline Buttons）进行渲染转换。
- **信息展示**：推荐“四段式”结构（标题、主体、详情、脚注），使用 Emoji 增强视觉警示效果。
- **按钮设计**：统一使用“查看详情”按钮，链接逻辑需支持自动登录重定向。
